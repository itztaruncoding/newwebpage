# MarketyDeals

A full-featured digital marketing deals & affiliate programs website built with React + Vite + Tailwind CSS.

## Features

- 🏠 Rich landing page with all sections
- 📰 20 blog posts with full articles
- 💰 6 deals with countdown timers and detail pages
- 🎨 Smooth Framer Motion animations
- 📱 Fully responsive (mobile + desktop)
- ☀️ Light mode design

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion
- Wouter (routing)
- shadcn/ui components
- react-icons
